# using Coingecko API
using the api coingecko to pull data from coingecko.com and filter them

## installation
PyPI

```
pip install pycoingecko
```
or from source

```
git clone https://github.com/man-c/pycoingecko.git
cd pycoingecko
python3 setup.py install
```
